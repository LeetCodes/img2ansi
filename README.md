# img2ansi

Converts an image to ANSI (using SGR escape sequences)

## Usage

```shell
img2ansi --help
# Usage of img2ansi: [options] <image>
#   -o string
#         Output file
#   -w int
#         Width of ANSI image (default 100)
```

## Demo

```shell
img2ansi examples/gopher.png
```
